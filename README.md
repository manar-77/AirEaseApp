# AirEase

AirEase is a smart airport companion mobile application built using React Native (Expo).  
It is designed to assist travelers by providing real-time flight information, accessibility features, and navigation tools within the airport.

## Features

- Flight Tracking  
  Search for flights and view details such as:
  - Status  
  - Gate  
  - Terminal  
  - Schedule  

- Luggage Tracking (BLE)  
  Bluetooth Low Energy functionality to simulate smart luggage tracking.

- Sign Language Support  
  Predefined phrases with GIFs and Text-to-Speech to assist communication.

- Airport Map  
  Simple map interface with gate navigation.

- Voice Assistant (Demo)  
  Voice-based navigation between app screens using predefined commands.

- Accessibility Features  
  - High contrast mode  
  - Large text support  

## Tech Stack

- React Native (Expo)  
- TypeScript  
- Firebase  
- BLE (react-native-ble-plx)  
- Expo APIs (Camera, Location, Notifications, Text-to-Speech)

## Project Idea

AirEase is a prototype focused on improving the airport experience, especially for users who require accessibility support. It combines flight tracking, luggage tracking, and voice assistance into a unified mobile solution.
